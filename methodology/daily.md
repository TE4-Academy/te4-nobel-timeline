# Daily Standup - Nobel Quest

## Syfte

Ett kort dagligt möte (max 15 minuter) där teamet synkar status och planerar dagen.

**Mål:**
- Alla vet vad alla gör
- Blockers identifieras tidigt
- Team känner framsteg

## Format

**När:** Varje dag kl 09:00
**Längd:** Max 15 minuter
**Deltagare:** Alla teammedlemmar, Project Lead faciliterar

## Struktur

Varje person svarar på 3 frågor:

1. **Vad gjorde jag igår?**
2. **Vad ska jag göra idag?**
3. **Har jag några blockers?**

**Exempel:**
```
Johan (UI/UX):
- Igår: Skapade game card design med Tailwind
- Idag: Implementera responsive layout för mobil
- Blockers: Inga

Lisa (Game Logic):
- Igår: Började på timer logic, 50% klar
- Idag: Slutföra timer och testa edge cases
- Blockers: Osäker på hur score-calculation ska fungera

Ahmed (Project Lead):
- Igår: Review av PR och merge
- Idag: Skapa Issues för dag 2, testa spelet
- Blockers: Inga
```
## Hantera blockers

Om någon har blockers:
- Diskutera INTE lösningen under standup
- Boka tid efter standup: "Kan du hjälpa mig kl 10:00?"
- Project Lead antecknar blockers och följer upp

## Tips

**Var konkret:**
Säg "Implementerade timer logic", inte "Jobbade på koden"

**Var kortfattad:**
Max 1-2 minuter per person

**Var ärlig:**
Om du är blockad, säg det direkt

**Håll tiden:**
Sätt timer på 15 minuter

**Lös problem senare:**
Standup är inte för problemlösning

## Checklista

### Project Lead
- [ ] Starta på tiden
- [ ] Alla får prata
- [ ] Anteckna blockers
- [ ] Uppdatera GitHub Projects efter standup

### Teammedlemmar
- [ ] Förberett vad jag ska säga
- [ ] Kort och koncist
- [ ] Ärlig om blockers

## Efter standup

- Project Lead uppdaterar GitHub Projects board
- Folk med blockers bokar follow-up
- Alla börjar jobba på dagens tasks
