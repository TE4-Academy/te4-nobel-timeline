# Roll: Project Lead

## Ansvar

Du ansvarar för projektets framsteg och att teamet samarbetar effektivt. Du håller koll på vad som görs, testar färdiga features och ser till att kod kommer in i projektet på rätt sätt.

## Dagliga uppgifter

### Morgon
- Facilitera daily standup (15 minuter)
- Uppdatera GitHub Project board
- Kolla igår kvällens commits

### Under dagen
- Testa färdiga features innan merge
- Hjälp teamet om någon fastnar
- Håll Issues uppdaterade

### Eftermiddag
- Verifiera att dagens arbete är pushat
- Planera imorgon med teamet

## Huvudsakliga leveranser

- Skapa och uppdatera GitHub Issues
- Håll GitHub Project board aktuellt
- Review och merga Pull Requests
- Testa spelet kontinuerligt
- Se till att README är uppdaterat

## Samarbete

Du pratar med alla i teamet dagligen. Hjälp till när någon fastnar. Testa det som andra bygger. Merga kod när den är klar och testad.

## Verktyg

- GitHub Projects (Kanban board)
- GitHub Issues och Pull Requests
- Browser för testing
- Git för branches och merging

## Checklista varje dag

- [ ] Daily standup genomförd
- [ ] GitHub Project board uppdaterat
- [ ] Pull Requests reviewade
- [ ] Nya features testade
- [ ] Dagens arbete pushat

## Vanliga utmaningar

**"Vi hinner inte allt"**
Fokusera på MVP först. Nice-to-have kan väntas.

**"Någon är blockerad"**
Hjälp direkt eller omfördela arbetet.

**"Merge konflikter"**
Merga från main ofta. Kommunicera vem som jobbar var.
